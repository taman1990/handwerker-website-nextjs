# Elektro Müller – Mobile-First Business Website

A realistic, mobile-first business website built with **Next.js**, **TypeScript**, and **Tailwind CSS**.

Designed to reflect a real German small-business site with focus on:
- mobile usability
- clean layout hierarchy
- trust & credibility
- accessibility basics

## Features
- Mobile-first responsive design
- Sticky header with scroll-based height reduction
- Active navigation highlighting on scroll
- Conversion-focused hero & contact form
- Reusable layout components (Section, Container)
- Clean, maintainable code structure

## Tech Stack
- Next.js (App Router)
- TypeScript
- Tailwind CSS

## Purpose
Portfolio project demonstrating real-world frontend design decisions for small business websites.

_Not a template. Not a UI demo. Built to feel like a real client project._

This project is part of a personal portfolio.
