# Handwerker Website (Next.js)

![Handwerker Website Screenshot](./public/handwerker-website-nextjs.png)

A **realistic, mobile-first website** for a German local service business.  
Built with **Next.js**, **TypeScript**, and **Tailwind CSS v4**.

This project focuses on how real small-business websites in Germany are structured, designed, and shipped — not on UI experiments or templates.

---

## Project Focus

- Mobile-first usability
- Clear content hierarchy
- Trust & credibility signals (German context)
- Accessibility basics
- Calm, professional design language

Designed to feel **production-ready**, not conceptual.

---

## Features

- Mobile-first responsive layout
- Sticky header with compact height
- Conversion-focused hero section
- Services, trust, testimonials, and contact sections
- GDPR-friendly contact flow
- Reusable layout primitives (`Section`, `Container`)
- Design-token–based styling (Tailwind v4)

---

## Tech Stack

- **Next.js** (App Router)
- **TypeScript**
- **Tailwind CSS v4** (CSS-first theming)

---

## Purpose

This is a **portfolio project** demonstrating real-world frontend and layout decisions for small business websites.

- No fake branding
- No over-engineering
- No UI-only mockups

Built to reflect how a real client project would be implemented and maintained.

---

## Notes

- Business name and content are fictional
- Legal pages (Impressum & Datenschutz) are included for structural completeness
- No backend or form handling included
